# Gerenciador de Pesquisas Clínicas
Gerenciador de Banco de Dados de Pesquisas Clínicas - SCC0640 - Base de Dados

## Rodando o Programa
Após clonar o repositório, dentro de um terminal, vá até a pasta clonada Trabalho-BD e rode o comando python3 -m MainProgram.main. Isso fará o programa se iniciar.

<b>OBS:</b> se estiver usando linux, especialmente arch, possivelmente será necessário o uso de um ambiente virtual devido aos pacotes python. Nesse caso, para inicializar o programa é necessáario, novamente, estar dentro de TrabalhoBD, porém utilizar o python desse ambiente, utilizando os comandos:

``` bash
    source venv/bin/activate
    ~/venv/bin/python -m MainProgram.main
```
